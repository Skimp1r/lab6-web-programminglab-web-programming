# ЛР6 — Дополнительное задание (6.2.3)

Реализован учет рабочего времени активности пользователя на сайтах:
- активность фиксируется по событиям `mousemove`, `keydown`, `click`;
- статистика по сайтам хранится в IndexedDB (в service worker расширения);
- при нажатии на кнопку расширения отображается статистика текущего сайта и топ активных сайтов.

Также сохранены функции предыдущего этапа (6.2.2):
- форма личных данных студента;
- сравнение документов;
- Google Drive / локальный fallback для отчета.
