# Лабораторная работа 6 — основное задание

## Что реализовано
- Расширение Chrome:
  - страница загрузки документов (`.doc/.docx`) с drag-and-drop;
  - поля личных данных студента: Ф.И.О., курс, факультет, направление подготовки, группа;
  - страница списка загруженных документов;
  - хранение документов, личных данных и результатов сравнения в `IndexedDB`.
- Node.js сервер:
  - endpoint `POST /api/compare`;
  - извлечение текста из `docx` (`mammoth`) и `doc` (`word-extractor`);
  - сравнение текстов алгоритмом шинглов (Jaccard similarity);
  - отправка загруженного файла в Google Drive;
  - создание текстового отчёта (данные студента + результаты сравнения) и загрузка в Google Drive;
  - fallback: если Google Drive недоступен, файл и отчёт сохраняются локально в `server/reports/<ФИО>/`.
- ВКонтакте:
  - на страницах `vk.com` рядом со ссылками на `doc/docx` добавляется кнопка `проверить`.

## Структура
- `server/` — серверная часть Node.js.
- `extension/` — расширение Chrome (Manifest V3).

## Запуск сервера
1. Открой терминал в `server`.
2. Выполни:
   - `npm install`
   - `npm start`
3. Сервер поднимется на `http://localhost:3000`.

## Настройка Google Drive
1. Создай Google Cloud Service Account с доступом к Google Drive API.
2. Скачай JSON ключ сервисного аккаунта.
3. Создай переменные окружения:
   - `GOOGLE_SERVICE_ACCOUNT_PATH` — путь к JSON ключу;
   - `GOOGLE_DRIVE_ROOT_FOLDER_ID` — (опционально) ID корневой папки на Google Drive.
4. Если указан `GOOGLE_DRIVE_ROOT_FOLDER_ID`, для каждого студента папка создаётся внутри неё.
   Если не указан — папка создаётся в корне Drive сервисного аккаунта.

## Установка расширения в Chrome
1. Открой `chrome://extensions/`.
2. Включи `Режим разработчика`.
3. Нажми `Загрузить распакованное расширение`.
4. Выбери папку `extension`.

## Проверка
1. Открой расширение и перетащи `.doc` или `.docx`.
2. Заполни личные данные студента.
3. Нажми `Проверить`.
4. Проверь, что документ появился в списке и есть проценты совпадения с ранее загруженными файлами.
5. Проверь в Google Drive, что создана папка с Ф.И.О. студента, загружен исходный файл и `.txt` отчёт.
6. Открой страницу `vk.com` со ссылкой на `doc/docx` — рядом должна быть кнопка `проверить`.
